## Requirements

-   Docker
-   npm
-   VS Code

Setup:

1. Clone the app and install its dependencies:

    ```shell
    git clone https://github.com/pelicanapp/Pelican
    cd Pelican
    npm i
    ```

2. Create a `pelican.json` file containing app secrets:

    ```shell
    npm run setup
    ```

Running the server locally:

```shell
npm run dev
```
